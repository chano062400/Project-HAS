# Project HAS

### 소개

#### 언리얼 엔진에서 제공하는 GameplayAbilitySystem을 이용하여 제작한 핵앤슬래시 장르의 개인 게임 프로젝트

- **레벨 업으로 얻는 포인트를 사용하여 스텟, 스펠 업그레이드 시스템.**
- **3 가지 속성 총 9 가지의 다양한 스킬을 조합한 전투.**
- **능력치를 올려주는 아이템을 관리하는 인벤토리 시스템.**
- **Listen Server 를 이용한 멀티 플레이 구현**
