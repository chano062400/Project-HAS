# Project HAS

### 소개

#### 언리얼 엔진에서 제공하는 GameplayAbilitySystem을 이용하여 제작한 핵앤슬래시 장르의 개인 게임 프로젝트

- **전투를 통해 성장하는 RPG 게임**
- **나만의 스킬 트리를 조합해 다양한 스킬을 이용한 전투**
- **Diablo4를 레퍼런스로 제작.**
---
### 개요

#### 제작 기간 - 2024년 5월 23일 ~ 2024년 7월 19일(약 8주)

#### 프로젝트 참여 인원 - 개인 제작

#### 활용 스킬 - C++, Unreal Engine5, GAS 프레임워크
